# 自用小工具
## KeyIntervalMonitor-Live
一个用来查看按键间隔的小浮窗工具。
打开后会显示在屏幕右上角，实时监测键盘按键之间的时间差。比如先按 W，再按 Shift，窗口里会显示 Shift-W 以及中间间隔的毫秒数。继续按其他键时，会继续更新最近的按键间隔，最多保留 3 条记录。
适合用来观察游戏里的连招、急停、跳投、身法操作，或者单纯测试自己两个按键之间的反应时间。
工具使用 Raw Input 读取键盘事件，适配 CS2 这类游戏场景，不需要注入游戏，也不会修改游戏文件。

## CS2ShiftEnglish
一个给 CS2 用的小后台工具。
运行后会常驻在系统托盘里。当检测到 cs2.exe 成为当前前台窗口时，它会自动模拟轻按一次 Shift，用于把输入法切到英文状态，避免进游戏后按键时弹出中文输入法候选框。
它只在切回 CS2 的时候触发一次，不会一直重复按 Shift。右键托盘图标可以退出。
如果 CS2 是管理员权限运行，这个工具也建议用管理员权限打开。
